Epic Filters

NeRF Data Filters for EPIC Kitchens Dataset

Description

This repository provides filters for NeRF data preparation for the EPIC Kitchens Dataset as of 2023. It utilizes the Epic Fields JSON files which are well-prepared. These filters can be tuned to meet your specific needs.

Update: An unzipper Python script has been added, which extracts and cleans all the EPIC Kitchens data. A processed minimal RGB frames dataset for NeRF training is coming soon.

References

  1. JSON files from the EPIC Fields dataset
  2. Dataset from EPIC Kitchens

Requirements (as of 2023)

  1. Clone this repository: git clone https://github.com/1ssb/epic-filters
  2. Install Python 3.10
  3. Install Libraries: pip install ultralytics cv2 opencv-python Pillow tqdm
  4. Recommended to use a CUDA-enabled GPU (Tested on an RTX 4090)

Note: If you would like to use the CPU, it's possible to modify the code. It's recommended to use concurrency-based Lock methods for the transformations in Overlap-filter.
